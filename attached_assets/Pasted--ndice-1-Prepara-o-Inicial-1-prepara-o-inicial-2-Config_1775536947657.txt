Preparação Inicial ### ✅ Checklist de Pré-requisitos Antes de começar, certifique-se de ter instalado: - [ ] **Node.js** versão 18 ou superior   - Testar: node --version deve mostrar v18.0.0 ou superior   - Download: https://nodejs.org/ - [ ] **Python** versão 3.9 ou superior   - Testar: python --version ou python3 --version   - Download: https://www.python.org/ - [ ] **Google Chrome** instalado   - Qualquer versão recente serve - [ ] **Telegram** instalado no celular   - Download: https://telegram.org/ --- ## 2. Configurar Telegram ### 2.1 Criar o Bot 1. **Abra o Telegram** no celular ou desktop 2. **Procure por @BotFather** na busca 3. **Clique em INICIAR** quando abrir a conversa 4. **Envie o comando:**    
/newbot
5. **Escolha um nome** para seu bot (exemplo: "Meu Monitor de Roleta") 6. **Escolha um username** (deve terminar com "bot", exemplo: "meu_monitor_roleta_bot") 7. **Copie o TOKEN** fornecido    - Formato: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz    - ⚠️ **IMPORTANTE**: Guarde este token com segurança! ### 2.2 Obter o Chat ID 1. **Procure por @userinfobot** no Telegram 2. **Clique em INICIAR** 3. **Copie o número** que ele mostrar    - Será algo como: 123456789    - Este é o seu **Chat ID** ### 2.3 Testar o Bot 1. **Procure seu bot** pelo username que você criou 2. **Clique em INICIAR** para começar a conversa --- ## 3. FAQ ### ❓ O backend não inicia **Problema:** Erro ao executar python app.py **Soluções:**
bash
# 1. Verifique se está no ambiente virtual
# Deve aparecer (venv) no início da linha do terminal

# 2. Reinstale as dependências
pip install --upgrade -r requirements.txt

# 3. Verifique a porta 5000
# Windows:
netstat -ano | findstr :5000
# Mac/Linux:
lsof -i :5000
### ❓ Frontend não conecta ao backend **Problema:** "Backend offline - Usando modo simulação" **Soluções:** 1. Verifique se o backend está rodando em http://localhost:5000 2. Teste manualmente: abra http://localhost:5000/api/ping no navegador 3. Verifique o arquivo .env:    
VITE_API_URL=http://localhost:5000
4. Reinicie o frontend (npm run dev) ### ❓ Telegram não recebe mensagens **Problema:** "Falha ao testar conexão" **Soluções:** 1. ✅ Verifique se o token está correto (sem espaços extras) 2. ✅ Verifique se o Chat ID está correto 3. ✅ Certifique-se de que iniciou uma conversa com o bot 4. ✅ Teste o token manualmente:    
bash
   curl "https://api.telegram.org/bot<SEU_TOKEN>/getMe"
### ❓ Selenium não conecta ao Chrome **Problema:** Erro ao executar monitor_real.py **Soluções:** 1. ✅ Feche TODOS os Chromes abertos 2. ✅ Abra o Chrome com o comando correto (veja seção 5.2) 3. ✅ Verifique se a porta 9222 está livre:    
bash
   # Windows:
   netstat -ano | findstr :9222
   # Mac/Linux:
   lsof -i :9222
4. ✅ Instale o Selenium corretamente:    
bash
   pip install selenium webdriver-manager
### ❓ Seletor CSS não encontra o número **Problema:** Monitor não detecta números novos **Soluções:** 1. ✅ Verifique se o seletor está correto (use DevTools) 2. ✅ A página pode ter mudado, encontre o novo seletor 3. ✅ Exemplos comuns de seletores:    
[data-testid="single-result"]
   .roulette-number
   #current-number
   .last-result span
4. ✅ Adicione prints para debug no monitor_real.py:    
python
   print(f"Seletor: {self.number_selector}")
   print(f"Elementos encontrados: {len(elements)}")
